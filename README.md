# 🎓 Funstone Preschool Website

🌐 Live Website: https://funstone.netlify.app/

A modern, responsive, and user-friendly preschool website built for **Funstone Preschool**.  
This website is designed to showcase school information, programs, and collect admission inquiries online.

---

## ✨ About Funstone Preschool

Funstone Preschool focuses on building a strong foundation for young learners through fun-based and activity-based learning.  
The curriculum is inspired by modern teaching methods like Montessori, Play-way, and experiential learning to support overall child development. :contentReference[oaicite:0]{index=0}  

---

## 🚀 Features

- 🏫 Beautiful and colorful preschool UI
- 📸 Gallery section for activities & classroom
- 📝 Admission form with Supabase integration
- 📱 Fully responsive (mobile-friendly)
- 📞 Contact section with WhatsApp integration
- ⚡ Fast performance (deployed on Netlify)

## 🛠️ Tech Stack

- **Frontend:** Antigravity / Next.js  
- **Backend:** Supabase  
- **Database:** PostgreSQL (via Supabase)  
- **Hosting:** Netlify  

---

## 🔗 Supabase Integration

The admission form is connected to Supabase to store student inquiry data.

### Stored Fields:
- Child Name  
- Age  
- Parent Name  
- Phone Number  
- Created At
